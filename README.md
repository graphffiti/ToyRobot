# ToyRobot

## How to build
1. Go to your working directory
2. git clone https://github.com/graphffiti/ToyRobot.git
3. cd ToyRobot
4. cmake .
5. make all


## How to run
./ToyRobot

## Requirements
This program uses C++17 Standard
